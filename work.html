SVC CS 143: Chess, implemented in Java

Created as a final project, this was a functional game of chess implemented using Java. Features include:

  • Piece and Boardstate storage
  • Movement logic for each piece
  • Path blocking to prevent illegal moves
  • Calculation of legal moves per piece
  • Capturing of enemy pieces
  • Basic enemy behaviour programming
  • User piece selection and movement
